3주차: VAE, Sampler, Scheduler 깊이 파기

1~2주차에서 "노드를 연결하면 이미지가 나온다"는 것을 경험했다면, 이번 주는 왜 그 결과가 나오는지를 이해하는 시간이다. KSampler 안에 숨어 있는 파라미터를 하나씩 분해하고, 의도대로 제어할 수 있게 된다.

학습 목표
  • ● VAE Encode/Decode의 역할과 Latent Space 개념을 설명할 수 있다
  • ● Sampler와 Scheduler의 차이를 구분하고, 상황에 맞는 조합을 선택할 수 있다
  • ● CFG Scale과 Steps가 결과에 미치는 영향을 예측할 수 있다
  • ● XY Plot을 활용하여 파라미터를 체계적으로 비교할 수 있다
  • ● 자신의 작업 스타일에 맞는 "나만의 기본 프리셋"을 설정할 수 있다

● CLIP 심화 — 토큰 제한과 CLIP Skip ▶

CLIP(Contrastive Language-Image Pre-training)은 텍스트와 이미지를 같은 공간에서 이해하는 모델이다. 프롬프트가 구체적일수록 더 명확한 벡터를 생성한다.

대략적 토큰 수 (실제 CLIP 토크나이저와 다를 수 있음) 0 / 77
  • ● 토큰 제한: CLIP은 한 번에 약 77개 토큰만 처리. 긴 프롬프트는 뒷부분이 잘린다
  • ● 단어 순서: CLIP은 단어의 순서보다 각 단어의 존재 여부에 더 민감
  • ● CLIP Skip: CLIP의 마지막 레이어를 건너뛰는 기법. Skip 2를 쓰면 더 추상적/예술적 결과 (애니메이션 스타일에서 자주 사용)

01 VAE — Latent Space의 출입구

VAE(Variational Autoencoder)는 이미지 ↔ Latent 변환기다. Stable Diffusion은 픽셀을 직접 다루지 않고 압축된 Latent Space에서 노이즈를 제거하며 작업한다.

VAE 파이프라인 — 클릭하여 데이터 흐름 확인
Noise 랜덤 Latent
KSampler 노이즈 제거
VAE Decode Latent → Pixel
Image 최종 출력
핵심 비유
VAE Encode는 zip 압축, VAE Decode는 zip 해제와 같다. Stable Diffusion은 이 압축된 공간(Latent Space)에서 작업하기 때문에 빠르고 효율적이다.

VAE 종류

구분 설명 사용 시점
내장 VAE 체크포인트에 포함된 VAE 기본값. 대부분의 경우 충분
외부 VAE 별도 VAE 파일 로드 (예: vae-ft-mse-840000) 색감이 탁하거나 디테일이 부족할 때
VAE Decode (Tiled) 타일 단위로 분할 디코딩 VRAM 부족 또는 고해상도 처리 시
02 Sampler — 노이즈를 제거하는 알고리즘

Sampler는 노이즈가 가득한 Latent에서 이미지를 "발견"해내는 알고리즘이다. 같은 프롬프트, 같은 시드라도 Sampler에 따라 결과가 달라진다.

03 Scheduler — 노이즈 제거의 속도 곡선

Scheduler는 각 스텝에서 노이즈를 얼마나 제거할지의 스케줄(시간표)을 결정한다. Sampler가 "어떻게"라면, Scheduler는 "얼마만큼씩"을 정한다.

● Karras가 인기 있는 이유 ▶

Normal이 각 스텝마다 동일하게 노이즈를 제거한다면, Karras는 초반에 대략적 구조를 잡고, 후반에 세밀한 디테일에 집중한다.

  • ● Normal: 30스텝 필요한 품질을 → Karras: 20~25스텝으로 도달
  • ● 생성 속도 차이는 5~10% 정도로 미미
추천 조합
조합용도
DPM++ 2M + Karras가장 범용적. 일상 생성의 기본값으로 추천
Euler + Karras빠르고 선명. 테스트용
DPM++ SDE + Karras최고 품질. 최종 출력용
Euler a + Normal창의적 변형 탐색용

CFG(Classifier-Free Guidance) Scale은 생성 이미지가 프롬프트를 얼마나 충실히 따를지를 결정한다.

영상 디자이너 팁
톤/무드 작업에는 CFG 5~7이 자연스러운 컬러 그레이딩 효과를, 그래픽 요소가 강한 작업에는 CFG 8~10이 효과적이다.
05 Steps — 반복 횟수와 품질의 관계

Steps는 노이즈 제거를 몇 번 반복할지를 정한다. 많을수록 좋은 것이 아니라, 수확 체감 지점이 있다.

Steps별 품질 변화 — 슬라이더를 움직여보세요
25

일반적 권장 범위. 품질과 속도의 최적 균형. 대부분의 생성에 적합하다.

실전 전략: 2단계 워크플로우
탐색 단계: 10~15 Steps + Euler a → 빠르게 100장 생성, 마음에 드는 시드 선별
완성 단계: 선별한 시드로 25~30 Steps + DPM++ 2M Karras → 고품질 출력

전부 고스텝으로 생성하는 것보다 3~8배 빠르면서 동일한 최종 품질을 얻을 수 있다.
