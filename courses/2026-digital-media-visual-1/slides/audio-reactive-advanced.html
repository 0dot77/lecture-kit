TOP Review

TOP 핵심 복습

TOP = Texture Operator — 이미지(텍스처)를 생성하고 처리하는 노드

입력

카메라, 영상, 이미지, 노이즈 등 시각 소스를 만든다

처리

합성, 변형, 색 보정, 왜곡 등 이미지를 가공한다

소스 TOP → 처리 TOP → 처리 TOP → 출력 / Null

Feedback TOP

Feedback Loop 원리

잔상과 누적 — 이전 프레임의 결과가 다음 프레임의 입력이 된다

Feedback → Composite → Level → Null

Level TOP의 Opacity를 반드시 1보다 작게 설정 — 그렇지 않으면 화면이 하얗게 폭발한다!

Feedback의 Target TOP을 마지막 Null로 지정하면 루프가 완성된다

Feedback + Transform

Feedback + Transform = 무한 터널

핵심 트릭

Feedback 루프 안에 Transform TOP을 넣고:

  • Scale: 0.98 (조금씩 축소)
  • Rotate: 1 (천천히 회전)

결과

이미지가 매 프레임 축소 + 회전되며 무한히 빨려들어가는 터널 효과가 만들어진다

Feedback → Transform → Composite → Level → Null

Displacement 원리

Source (원본) → Displace TOP ← Displace Image (노이즈 등)

Timemachine TOP

Timemachine TOP

Video Device In → Texture 3D → Timemachine ← Ramp (2nd input)

Texture 3D

시간순으로 프레임을 쌓아 3D 볼륨 텍스처를 만든다

Timemachine

Ramp의 밝기값으로 "어느 시점의 프레임을 볼지" 결정 — 시간 조작

Ramp를 수직으로 설정하면 화면 위 = 과거, 아래 = 현재

Useful Combos

오디오 리액티브에서 자주 쓰는 TOP 조합

Feedback + Level + Composite

소리에 따라 Opacity를 조절하면 잔상이 음악에 반응

Feedback + Displace + Noise

노이즈 강도를 볼륨에 연결하면 소리에 따라 이미지가 일렁인다

Texture3D + Timemachine + 오디오

오디오 값으로 시간축을 탐색하면 소리에 따라 과거 프레임이 나타난다

HSV Adjust + Level

주파수에 따라 색상(Hue)을 변경, 볼륨으로 밝기 조절

CHOP Review

CHOP 핵심 복습

CHOP = Channel Operator — 시간에 따라 변하는 숫자(채널)를 생성하고 가공하는 노드

인터랙션의 기본 구조

입력 (센서/오디오/마우스) → 가공 (필터/매핑/분석) → 출력 (TOP 파라미터)

CHOP은 숫자의 흐름이다. 소리, 마우스, LFO, 타이머 등 모든 변화하는 값을 다룬다.

Audio Device In

Audio Device In CHOP

소리 받기 — 마이크나 오디오 인터페이스에서 실시간 소리를 TouchDesigner로 가져온다

기본 설정

  • Device: 사용할 오디오 장치 선택
  • Driver: DirectSound / ASIO
  • Channels: Stereo (Left + Right)

출력

실시간 파형 데이터가 채널로 출력된다

MacOS: CoreAudio / Built-in Microphone

Raw Audio Signal

Audio Device In 출력 특성

채널

chan1 = Left
chan2 = Right

값 범위

-1 ~ +1
양수와 음수를 빠르게 오간다

문제점

원시 신호를 그대로 파라미터에 연결하면 너무 빠르게 떨린다

그래서 분석(Analyze)이 필요하다 — 원시 파형에서 의미 있는 숫자를 추출해야 한다

Volume Extraction

볼륨 추출 — Analyze CHOP

Audio Device In → Analyze (RMS Power) → Filter → Null

Analyze CHOP

Function: RMS Power

수천 개의 샘플을 하나의 숫자(볼륨)로 요약한다

RMS란?

Root Mean Square — 파형의 "평균 에너지"를 계산

출력: 0 (무음) ~ 1 (최대)

Analyze Functions

Analyze CHOP Function 비교

Function설명특성
RMS Power파형의 평균 에너지 (실효값)가장 안정적, 볼륨 추출에 최적
Maximum샘플 중 가장 큰 값피크 감지, 급격한 변화에 민감
Average샘플의 산술 평균오디오에선 0에 가까움 (음수+양수 상쇄)

Average는 오디오 파형에 적합하지 않다 — 양수와 음수가 상쇄되어 거의 0이 된다. RMS Power를 사용하자!

Range Mapping

Math CHOP — 범위 조절

From Range

입력값의 실제 범위

From: 0 ~ 0.3 # 실제 RMS 범위

To Range

원하는 출력 범위

To: 0 ~ 1 # 파라미터 범위
Analyze → Math (Range) → Filter → Level TOP (Opacity)

볼륨 0~0.3 → 밝기 0~1 으로 매핑하면, 작은 소리에도 충분히 반응한다

Frequency Analysis

주파수 분석 개념

저주파 (Bass)

20 ~ 250 Hz

킥 드럼, 베이스 기타, 서브베이스

중주파 (Mid)

250 ~ 4,000 Hz

보컬, 기타, 피아노 멜로디

고주파 (High)

4,000 ~ 20,000 Hz

하이햇, 심벌즈, 치찰음

같은 음악이라도 저음/중음/고음을 분리하면 각각 다른 비주얼 요소를 제어할 수 있다

Palette — audioAnalysis

audioAnalysis 컴포넌트

Palette > Tools > audioAnalysis — 드래그 한 번으로 주파수 분리 + 비트 감지를 자동 처리

연속값 (주파수 대역)

low — 저주파 (베이스, 킥)

mid — 중주파 (보컬, 멜로디)

high — 고주파 (하이햇, 심벌)

이벤트 (비트 감지)

kick — 킥 드럼 감지 (0/1)

snare — 스네어 감지 (0/1)

rhythm — 전체 리듬 감지

각 채널별 Thresh, Gain, Smooth 노브로 감도 조절 가능

How to Use

audioAnalysis 사용법

  1. Palette 브라우저를 연다 (Alt+L)
  2. Tools 폴더에서 audioAnalysis를 찾아 네트워크에 드래그
  3. 컴포넌트가 자동으로 오디오를 분석 — 노드를 클릭하면 스펙트럼이 보인다
  4. CHOP 출력에서 원하는 채널을 Select CHOP으로 골라 사용

왜 audioAnalysis인가? — Audio Spectrum → Select CHOP으로 직접 주파수를 자르려면 샘플 인덱스를 계산해야 한다. audioAnalysis는 이 과정을 자동 처리하고, 노브로 바로 감도를 조절할 수 있어 훨씬 실용적이다.

Frequency Mapping

주파수별 매핑 예시

대역매핑 대상효과
Bass (저음)Transform → Scale베이스 드럼에 맞춰 이미지가 커졌다 줄어든다
Mid (중음)Displace → Weight멜로디에 따라 이미지가 일렁인다

각 대역을 서로 다른 시각 파라미터에 연결하면 음악의 구조가 시각적으로 드러난다

Beat Detection

비트 감지

Analyze (RMS) → Logic CHOP → Trigger CHOP → Null

Logic CHOP

Convert: Off When Outside Bounds

Bound Min/Max 범위 밖이면 0, 안이면 1

Trigger CHOP

1이 들어오면 Attack → Sustain → Release 엔벨로프 생성

부드러운 펄스로 변환

Threshold를 잘 조절하는 것이 비트 감지의 핵심 — 너무 낮으면 모든 소리에 반응, 너무 높으면 반응하지 않음

Practice 1

볼륨 리액티브 — 소리로 숨 쉬는 영상

소리의 볼륨(RMS)을 추출하여 영상의 밝기와 크기를 제어한다

목표

소리가 커지면 영상이 밝아지고 커진다. 소리가 작아지면 어두워지고 줄어든다.

사용 Operator

Audio Device In, Analyze, Math, Filter, Level TOP, Transform TOP

Practice 1 — Network

볼륨 리액티브 네트워크

Audio Device In → Analyze (RMS) → Math (Range) → Filter → Null

단계별 설정

  1. Audio Device In — Device 선택 (마이크 또는 시스템 오디오)
  2. Analyze — Function: RMS Power
  3. Math — From Range: 0~0.3 / To Range: 0~1
  4. Filter — Width: 0.1 (부드럽게)
  5. Null → Level TOP의 Opacity에 Export 또는 Reference
  6. 같은 값으로 Transform TOP의 Scale에도 연결 (0.8 ~ 1.2)
Practice 2

주파수 리액티브 — 저음/고음 분리

주파수 대역을 분리하여 저음과 고음을 각각 다른 비주얼 요소에 매핑한다

목표

베이스 드럼 → 이미지 스케일 변화
하이햇 → 노이즈 왜곡

사용 Operator

Audio Spectrum, Select, Analyze, Math, Displace TOP, Noise TOP

Practice 2 — Network

주파수 리액티브 네트워크

Audio Device In → Audio Spectrum → Select (저음) → Analyze → Transform Scale
Audio Device In → Audio Spectrum → Select (고음) → Analyze → Displace Weight

핵심 설정

Select CHOP의 Channel Name Range로 저음/고음 대역 분리 → 각각 Analyze (RMS) → Math로 범위 조절 → 서로 다른 TOP 파라미터에 연결

Practice 3

비트 반응 + Feedback

비트를 감지하여 Feedback 루프의 파라미터를 순간적으로 변화시킨다

목표

드럼 비트마다 Feedback 잔상이 강해지고, 비트 사이에는 서서히 사라진다

사용 Operator

Logic, Trigger, Feedback TOP, Level TOP, Composite TOP

Practice 3 — Network

비트 반응 + Feedback 네트워크

Analyze (RMS) → Logic (Threshold) → Trigger (A/S/R) → Null
Feedback → Level (Opacity ← Trigger) → Composite → Null (Target)

단계별 설정

  1. Logic — Channel Pre OP: Off to On, Boundary Threshold: 0.15
  2. Trigger — Attack: 0.02s, Sustain: 0.05s, Release: 0.3s
  3. Trigger 출력 → Level TOP Opacity에 Reference
  4. Feedback Target을 마지막 Null로 지정
