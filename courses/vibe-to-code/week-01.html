04 Block 2
Claude 에코시스템

Anthropic이 만든 AI. 단순 챗봇이 아닌 다양한 형태로 존재합니다.

  • ● 웹 브라우저에서 바로 사용
  • ● 프로젝트 기능: 맥락을 유지하는 대화
  • ● Artifacts: 대화 중 만들어지는 결과물 (코드, 문서, 다이어그램 등)
이렇게 해보세요 무슨 일이 일어나는지
"이 관계를 다이어그램으로 보여줘" 플로우차트/마인드맵이 만들어짐
이미지를 드래그해서 넣기 사진을 보고 분석하거나, 그걸 기반으로 만들어줌
"이걸 인터랙티브하게 만들어줘" 클릭/드래그 가능한 웹앱이 바로 실행됨
"내 응답 스타일을 이렇게 바꿔줘" 톤과 형식을 커스터마이즈
프로젝트에 파일 첨부 대화마다 반복 설명 안 해도 맥락 유지

같은 대화창인데, 어떻게 말하느냐에 따라 전혀 다른 결과가 나온다.

  • ● 터미널(명령줄)에서 동작하는 AI 에이전트
  • ● 대화로 파일을 만들고, 수정하고, 실행한다
  • ● "채팅"이 아니라 "함께 작업"하는 느낌

06 Install
Claude Code 설치하기

Claude Code는 Node.js가 필요합니다. 터미널에서 아래 명령어로 설치 여부를 확인하세요.

node --version ← v18 이상이 나오면 OK

설치가 안 되어 있다면:

운영체제 설치 방법
Mac 방법 1) nodejs.org에서 LTS 다운로드 & 설치
방법 2) Homebrew 사용:
brew install node
Windows nodejs.org에서 LTS 다운로드 & 설치
설치 후 PowerShell을 다시 열어야 합니다
터미널 열기 (Mac: Terminal / Windows: PowerShell)
Node.js 설치 확인: node --version
Claude Code 설치
실행: claude
로그인 진행
npm install -g @anthropic-ai/claude-code ← 설치 claude ← 실행
명령어 하는 일
claude 기본 실행 — 파일 수정/명령 실행 전에 매번 허락을 물어봄
claude --dangerously-skip-permissions 허락 없이 바로바로 실행 (빠르지만 주의 필요)
claude --model sonnet 사용할 모델 지정
claude --resume 마지막 대화 이어서 하기
claude /help Claude Code 안에서 도움말 보기
07 Block 3
도구로서의 AI, 에이전트로서의 AI

대부분의 AI 사용: 질문 → 답변, 요청 → 결과물.
그런데 AI와 함께 기획할 수 있다면?

같은 요청, 다른 경험

요청: "간단한 자기소개 웹페이지를 만들어줘"

claude.ai로 만들면

  • Artifact로 코드가 나옴
  • 복사해서 저장해야 함
  • 대화 맥락 안에서 동작

Claude Code로 만들면

  • 내 컴퓨터에 파일이 생김
  • 바로 브라우저에서 열 수 있음
  • 내 프로젝트 폴더 안에서 동작
Claude가 확장되는 구조
  • CLAUDE.md — 프로젝트의 맥락을 Claude에게 알려주는 파일 → 2주차에서 직접 작성
  • Skills — Claude에게 특정 능력을 가르칠 수 있다 → 4주차에서 만들어봄
  • MCP — 외부 도구와 연결 (Figma, Notion 등) → 3주차에서 가능성 탐색

Claude는 고정된 도구가 아니라, 내가 설계하는 시스템의 일부가 될 수 있다.

작가에게 이것이 의미하는 것
  • 나만의 창작 시스템을 설계할 수 있다
  • AI가 내 취향, 톤, 규칙을 기억하고 따르게 할 수 있다
  • 코드를 모르더라도, 시스템을 사고할 수 있다
  • 4주 후: 다시 작업할 때 꺼내서 창작의 단초로 삼을 수 있는 나만의 창작 워크플로우
08 Block 4
실습: 두 가지 방식으로 만들어보기

나를 표현하는 인터랙티브 자기소개 페이지

클릭하거나, 마우스를 움직이거나, 스크롤하면 반응하는 — 인터랙티브한 자기소개 웹페이지를 만들어봅시다.

  • 이름을 클릭하면 소개가 펼쳐진다
  • 마우스를 따라 배경 색이 바뀐다
  • 좋아하는 것들을 카드로 만들고, 뒤집으면 이유가 나온다
  • 스크롤하면 타임라인처럼 나의 이야기가 흘러간다

정답 없음. 자유롭게. "인터랙티브"의 정의도 자유.

Step 1 — claude.ai (10분)

대화로 만들어보기 — Artifact에서 바로 인터랙션 확인

나를 표현하는 인터랙티브 웹페이지를 만들어줘. 나는 ___한 사람이야. 클릭하면 ___가 일어나게 해줘.
Step 2 — Claude Code (25분)

파일이 생기고, 수정하고, 브라우저에서 여는 경험

mkdir my-vibe && cd my-vibe claude --dangerously-skip-permissions # 같은 요청을 해보기 open index.html
