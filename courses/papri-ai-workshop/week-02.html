+ 강한 줄 수업 자료는 항상 목표, 준비물, 시간표, 실습 단계, 공유 질문 순서로 작성한다.

좋은 CLAUDE.md는 내 취향을 설명하는 문서가 아니라, AI가 결정을 내릴 때 참고할 기준표입니다.

  • ·완료 기준 — 무엇이 좋은 결과인가

THE LOOP
생성 → 관찰 → 가드레일

결과가
어긋날 때마다
헌법을
고친다.

프롬프트를 길게 덧붙이는 대신,
내 작업 세계의 헌법을 한 조항씩 보강합니다.
한 번 추가한 조항은 다음 작업에도 재사용됩니다.

  1. 1차 생성
    AI가 내 헌법을 어떻게 해석했는지 본다
  2. 관찰
    마음에 드는 것과 어긋난 것을 표시한다
  3. 가드레일 추가
    어긋난 지점을 CLAUDE.md의 새 조항으로 쓴다
  4. 2차 생성
    같은 요청을 다시 한다
  5. 비교
    새 조항이 결과를 어떻게 바꿨는지 확인

— 프롬프트로 때우기 "더 예쁘게 해줘"
+ CLAUDE.md 조항 방은 3D 박스 블록보다 방꾸미기 게임의 2D 일러스트처럼 보여야 한다.
— 프롬프트로 때우기 "성격이 더 보이게 해줘"
+ CLAUDE.md 조항 모든 주요 오브제는 AI Character의 행동 규칙 중 하나와 연결되어야 한다.
— 프롬프트로 때우기 "좀 단순하게 해줘"
+ CLAUDE.md 조항 씬에는 핵심 오브제 5–7개만 배치하고, 장식용 소품은 피한다.
— 프롬프트로 때우기 "구도 다시 잡아줘"
+ CLAUDE.md 조항 PixiJS 결과물은 처음 로드 시 화면 안에 방 전체와 주요 오브제가 모두 보여야 한다.
